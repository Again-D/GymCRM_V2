## 한 줄 요약

GymCRM V2는 회원, 회원권, 예약, 출입, 라커, CRM, 정산을 한곳에서 처리하는 센터 운영 콘솔이다.
원문 매뉴얼에는 각 기능별 `업무 흐름`과 확인 포인트가 더 자세히 정리되어 있다.

## 누가 무엇을 보나

| 역할 | 자주 쓰는 화면 |
|---|---|
| 관리자/매니저 | 대시보드, 회원, 회원권, 예약, 상품, 트레이너, 출입, 라커, CRM, 정산, 감사 로그, 설정, 사용자 계정 관리 |
| 데스크/현장 직원 | 회원, 회원권, 예약, 상품, 트레이너, GX 스케줄, 출입, 라커, CRM |
| 트레이너 | 회원, 회원권, 예약, GX 스케줄, 정산, 내 스케줄, 내 계정 |

## 가장 먼저 익힐 흐름

1. `회원 관리`에서 회원을 찾는다.
2. `회원권 업무`에서 권한 상태를 확인한다.
3. `예약 관리`에서 오늘 예약을 처리한다.
4. 필요하면 `출입 관리`와 `라커 관리`를 본다.
5. `정산 리포트`로 운영 현황을 확인한다.

회원 관리에서는 이름/연락처/상태/회원권 상태/트레이너/상품 필터를 함께 쓴다.

## 자주 헷갈리는 점

- `회원 관리`는 사람을 찾는 곳이고, `회원권 업무`는 권한을 처리하는 곳이다.
- `예약`과 `출입`은 연결되어 있으므로 회원권 상태를 먼저 본다.
- `매출 분석`은 센터 전체 흐름, `트레이너 정산`은 개인 정산 흐름이다.
- `내 계정`은 본인 비밀번호 변경, `사용자 계정 관리`는 관리자의 계정 초기화 화면이다.
- 트레이너 관리와 내 스케줄은 월별 가용 캘린더를 함께 본다.

## 문제 발생 시 확인 순서

1. 현재 역할이 맞는지 본다.
2. 필터와 검색어를 초기화한다.
3. 회원권 상태를 확인한다.
4. 저장 오류면 메시지와 감사 로그를 함께 본다.
5. 비밀번호 문제면 `내 계정`인지 `사용자 계정 관리`인지 먼저 구분한다.
