# GYM CRM 요구사항
1. 회원관리
- 상단 검색 필터 추가 (트레이너 선생님별, 기간별, 상품별)
2. 예약 관리
- 예약권 보유하고 있는 회원들의 리스트화면 (검색필터 있어야 함)
1) 마스터 계정의 경우, 예약권 보유하고 있는 전체 회원 리스트 노출
2) 트레이너 선생님 계정의 경우, 선생님에게 할당된 회원 중 예약권을 보유하고 있는 사람들의 리스트 노출
- 회원명 클릭 시 회원 상세 모달 팝업 노출 (모달팝업에는 회원정보, 상품정보, 예약상태, 예약일정 포함)
- 예약일정에 대한 조정 기능 (노쇼,취소,과거 일정은 수정 불가, 현재 진행중이거나 앞으로 진행할 예약일정은 조정 가능하게)
